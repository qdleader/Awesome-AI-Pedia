## 🚀 快速开始

### 方法一：使用启动脚本（推荐）

```bash
# 克隆项目
git clone https://github.com/qdleader/Awesome-AI-Pedia.git
cd Awesome-AI-Pedia

# 给脚本添加执行权限（仅需执行一次）
chmod +x start.sh

# 运行启动脚本
./start.sh
```

脚本会自动：
- ✅ 检查 Node.js 环境
- ✅ 安装项目依赖
- ✅ 启动开发服务器
