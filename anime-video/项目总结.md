## 🚀 如何使用

### 1. 预览视频 (已启动)

Remotion Studio 已经在运行:
- 🌐 本地地址: http://localhost:3000
- 🖥️ 网络地址: http://172.26.6.224:3000

在浏览器中可以:
- ▶️ 实时预览视频
- 🎬 拖动时间轴查看不同场景
- ⚙️ 调整参数和配置
- 📊 查看性能指标

### 2. 渲染视频

在项目目录运行:

```bash
cd /Users/icourt/github/Awesome-AI-Pedia/anime-video

# 渲染为 MP4 格式
npm run build -- --codec=h264

# 或指定输出文件名
npm run build -- --codec=h264 --out=my-anime-video.mp4
```

视频将保存在 `out/` 目录中。

### 3. 自定义内容

#### 修改标题和副标题
编辑 `src/Root.tsx`:
```tsx
defaultProps={{
  title: "你的标题",
  subtitle: "你的副标题"
}}
```

#### 调整场景时长
编辑 `src/compositions/AnimeVideo.tsx`,修改各个 `<Sequence>` 的 `durationInFrames`。

#### 修改视觉效果
编辑各个场景组件:
- `src/components/TitleScene.tsx` - 标题场景
- `src/components/CharacterScene.tsx` - 角色场景
- `src/components/ActionScene.tsx` - 动作场景
- `src/components/EndingScene.tsx` - 结束场景
