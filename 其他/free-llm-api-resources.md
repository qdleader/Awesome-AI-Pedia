# 免费 LLM API 资源汇总 🚀

本页面整理了目前市面上主流且稳定的免费大模型（LLM）API 资源，旨在帮助开发者快速构建原型、进行模型测试及学习探索。

---

## 💎 核心推荐平台

以下平台提供的免费额度较为客观，且 API 稳定性较高：

### 1. [OpenRouter](https://openrouter.ai/)
- **说明**：极佳的模型聚合器。
- **免费资源**：支持 Llama 3、Mistral 等多种模型的免费调用。
- **优点**：统一的 OpenAI 兼容格式 API，切换模型极其简单。

### 2. [Google AI Studio (Gemini)](https://aistudio.google.com/)
- **说明**：Google 官方提供的开发平台。
- **免费资源**：提供 Gemini 1.5 Pro 和 Gemini 1.5 Flash 的免费层级。
- **限制**：通常有每分钟请求数（RPM）限制。

### 3. [NVIDIA NIM](https://build.nvidia.com/explore/discover)
- **说明**：英伟达推出的推理微服务。
- **免费资源**：新注册用户通常可获得充足的免费积分用于调用 Llama 3、Mixtral 等热门模型。

### 4. [Groq Cloud](https://console.groq.com/)
- **说明**：以超高速推理著称。
- **免费资源**：目前提供 Llama 3、Mixtral 等开源模型的免费推理 API。

### 5. [Hugging Face Inference API](https://huggingface.co/docs/inference-providers/en/index)
- **说明**：最大的开源模型社区。
- **免费资源**：支持数以万计开源模型的推理访问。

## 🏗️ 基础设施类

- **[Cloudflare Workers AI](https://developers.cloudflare.com/workers-ai)**：在边缘端提供多种开源模型的免费推理。
- **[Vercel AI Gateway](https://vercel.com/docs/ai-gateway)**：为 AI 应用提供增强的网关功能，包含免费层级。
