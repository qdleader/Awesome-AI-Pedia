# cli-anything 安装

在Claude Code上安装
# 添加 CLI-Anything 插件市场
/plugin marketplace add HKUDS/CLI-Anything

# 从市场安装 cli-anything 插件
/plugin install cli-anything
Claude Code 通过 bash 执行命令。Windows 下请安装 Git for Windows（包含 bash 和 cygpath） 或使用 WSL，否则可能出现 cygpath: command not found。

一行命令生成 CLI：

# /cli-anything:cli-anything <软件路径或仓库地址>
# 为 GIMP 生成完整的 CLI（7 个阶段全自动）
/cli-anything:cli-anything ./gimp
