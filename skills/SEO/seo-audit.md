# seo-audit

`seo-audit` 是由 `marketingskills` 仓库提供的一个强大的 AI Agent Skill，专门用于对网站进行技术性和页面内 (On-page) SEO 审计。它能够自动识别影响搜索排名的关键问题，并提供分优先级的优化建议。

---

## 🚀 快速开始

### 安装方式
在你的项目终端中运行：
```bash
npx skills add coreyhaines31/marketingskills --skill seo-audit
```
*注：该 Skill 兼容 Claude Code 等支持 Skill 系统的 AI 编程工具。*

### 使用方式
安装后，你可以通过以下指令或关键词触发：
- **核心命令**：`/seo-audit`
- **关键词触发**：当你在对话中提到 "SEO audit", "technical SEO", "why am I not ranking", "traffic drop" 等相关词汇时，AI 会自动调用该 Skill。

---

## 核心功能

1.  **技术性 SEO 审计**：检查索引问题、爬取错误、网站速度、移动端适配等。
2.  **页面内优化**：分析标题标签 (Title Tags)、元描述 (Meta Descriptions)、标题层级 (H1-H6) 等。
3.  **分级建议报告**：生成的报告会包含问题的严重性级别（高/中/低）、证据、修复建议以及改进的优先级。
