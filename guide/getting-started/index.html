快速开始指南 ​

欢迎使用 Awesome AI Pedia！本指南将帮助你快速了解如何使用本博客系统。

系统要求 ​

在开始之前，请确保你的系统已安装以下软件：

  • Node.js >= 18.0.0
  • npm >= 8.0.0 或 yarn >= 1.22.0
  • Git

本地运行 ​

1. 克隆仓库 ​

bash
git clone https://github.com/yourusername/awesome-ai-pedia.git
cd awesome-ai-pedia
1
2

2. 安装依赖 ​

bash
npm install
# 或
yarn install
1
2
3

3. 启动开发服务器 ​

bash
npm run dev
# 或
yarn dev
1
2
3

4. 访问网站 ​

打开浏览器访问 http://localhost:3000

添加新文章 ​

1. 创建文章文件 ​

在docs/blog/目录下创建新的Markdown文件：

bash
docs/blog/my-new-article/index.md
1

2. 编写文章 ​

在文章顶部添加Front Matter元数据：

markdown
---
title: 文章标题
date: 2024-01-26
author: 你的名字
readingTime: 5 分钟阅读
tags:
  - 标签1
  - 标签2
description: 文章描述
---

# 文章内容

在这里编写你的文章内容...

目录结构说明 ​

awesome-ai-pedia/
├── docs/                          # 文档根目录
│   ├── .vitepress/                # VitePress配置
│   │   ├── config.ts              # 主配置文件
│   │   ├── theme.ts               # 主题配置
│   │   ├── custom.css            # 自定义样式
│   │   └── components/            # 自定义组件

Front Matter字段说明 ​

字段类型说明必需
titlestring文章标题✅
datestring发布日期（YYYY-MM-DD格式）✅
authorstring作者姓名✅
readingTimestring预计阅读时间✅
tagsarray文章标签数组❌
descriptionstring文章描述✅
imagestring封面图片路径❌
